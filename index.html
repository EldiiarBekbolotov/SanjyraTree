Functionalities

Updated November 26, 2024.

Feature Description Purpose
New Member Allows users to create a new member profile with inputs for Name, Description, and Profile Picture. To add detailed information about family members or individuals for visualization.
Drag, Zoom, and Move Users can drag family members to rearrange them, zoom in/out for better visibility, and move across the editor canvas. To provide a dynamic and interactive way to customize the family tree layout.
Get Save Code Generates a unique save code representing the current state of the editor and family tree. To export and share data for future use or collaboration.
Load Save Code Enables users to input a previously generated save code to restore a saved family tree state. To easily recover and continue working on previously saved data.
Local Save Saves the app's data locally on the userâ€™s device using browser storage. To allow quick and reliable saving without external codes or connectivity.
Load Local Save Loads the locally saved data directly from the user's browser storage for seamless recovery. To provide a user-friendly way to continue from where they left off on the same device.
Responsive Design The app and editor are fully responsive, adapting to different screen sizes, browsers, and devices for optimal usability. To ensure accessibility and usability across a variety of platforms.

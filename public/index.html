Functionalities

Updated November 26, 2024.

Feature Description
New Member Enables users to create detailed profiles for family members by inputting a Name, Description, and uploading a Profile Picture. This functionality facilitates the visualization and organization of individuals within the family tree.
Drag, Zoom, and Move Offers an intuitive interface allowing users to rearrange family members through drag-and-drop functionality, zoom in and out for better visibility, and navigate across the editor canvas. This ensures a highly customizable and user-friendly experience.
Get Save Code Generates a unique save code representing the current state of the family tree editor. This feature makes it easy to export and share the tree structure for future use or collaborative purposes.
Load Save Code Input a previously generated save code to restore a saved state of the family tree. This ensures continuity and convenience when working across sessions or devices.
Local Save Save data directly to your device through browser storage. Saved on your computer until you clear your local storage.
Load Local Save Load previously saved data from your browser storage. Resume your work seamlessly on the same device.
Responsive Design SanjyraTree works in different devices (desktop computers, phone, and tablets), screen sizes, and browsers, ensuring an optimal user experience.
